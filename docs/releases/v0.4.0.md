# CountBot v0.4.0 发布说明

**发布日期**: 2026年3月13日  
**版本号**: 0.4.0  

---

## 🎉 核心亮点

### 1. 会话级自定义配置系统

每个会话现在都可以拥有独立的配置，让你构建不同特性的专属助理机器人：

- **独立 API 配置** - 每个会话可以使用不同的 LLM 提供商
- **独立模型选择** - 不同会话使用不同的模型（如 GPT-4、Claude、GLM-5）
- **独立提示词和人格** - 为每个会话定制专属的性格和行为
- **灵活切换** - 在会话面板中一键切换配置

**使用场景**：
- 会话 A：使用 Deepseek，专注文案创作
- 会话 B：使用 Claude，专注代码开发
- 会话 C：使用 GLM-5，专注中文对话

### 2. 新增三个 IM 渠道

#### 微博渠道
- 关注微博龙虾助手私信交互

#### 企业微信渠道
- 支持企业微信应用消息
- WebSocket 长连接

#### 小智 AI 渠道
- 支持小智 AI 平台集成



### 3. 全面优化多智能体协作

多智能体工作流系统得到全面增强：

- **条件判断增强** - 支持更复杂的工作流条件逻辑
- **依赖管理优化** - 智能体之间的依赖关系更清晰
- **错误处理改进** - 更好的异常捕获和恢复机制
- **性能提升** - 并发执行优化，减少等待时间

**三种协作模式**：
- **Pipeline 模式** - 顺序执行，适合流水线任务
- **Graph 模式** - 有向图执行，支持复杂依赖
- **Council 模式** - 投票决策，适合多方案评估

### 4. 新增 /help 命令

在任何 IM 渠道中输入 `/help` 即可查看可用命令：

```
可用命令：
/help - 显示帮助信息
/all - 查看所有会话
/new - 创建新会话
/switch <session_id> - 切换会话
/delete <session_id> - 删除会话
/config - 查看当前会话配置
```

### 5. 集成 Mermaid 图表渲染

前端现在支持 Mermaid 图表渲染，AI 可以生成各种图表：

- **流程图** - 业务流程、算法流程
- **时序图** - 系统交互、API 调用
- **甘特图** - 项目计划、任务安排
- **类图** - 系统架构、数据模型
- **状态图** - 状态机、生命周期
- **饼图** - 数据分布、占比分析

**示例**：
```
请帮我画一个用户登录的流程图
```

AI 会生成 Mermaid 代码，前端自动渲染为可视化图表。

### 6. 全面兼容 OpenClaw Skills 技能生态

CountBot 现在完全兼容 OpenClaw Skills 生态系统：

- **外部技能目录** - 支持 `~/.openclaw/skills` 目录
- **自动发现** - 启动时自动扫描并加载外部技能
- **跨平台路径** - Windows/macOS/Linux 路径兼容
- **技能创建器** - 新增 skill-creator 工具

**使用方式**：
1. 将 OpenClaw 技能放入 `~/.openclaw/skills/` 目录
2. 重启 CountBot 或热重载工作空间
3. 技能自动加载，无需额外配置

### 7. Heartbeat 主动问候系统重构

主动问候系统得到全面重构，更加智能和自然：

- **真随机算法** - 基于日期种子的真随机时间生成
- **状态持久化** - 问候状态保存到本地文件
- **免打扰优化** - 更精确的免打扰时段控制
- **每日限额** - 防止过度打扰（可配置）

**工作原理**：
```python
# 不是固定时间，而是在活跃时段内随机选择
# 例如：9:00-22:00 之间随机选择 2 个时间点
# 每天的时间点都不同，更加自然
```

## 📦 升级指南

### 从 v0.3.x 升级

```bash
# 1. 备份数据（可选但推荐）
cp backend/countbot.db backend/countbot.db.backup

# 2. 拉取最新代码
git pull origin main

# 3. 更新依赖
pip install -r requirements.txt

# 4. 重启服务
python start_app.py
```

数据库会自动迁移，无需手动操作。

### 全新安装

```bash
# 克隆仓库
git clone https://github.com/countbot-ai/countbot.git
cd countbot

# 安装依赖
pip install -r requirements.txt

# 启动
python start_app.py
```
