## 核心更新

### 1. 多智能体协作系统

实现了完整的多智能体工作流系统，支持三种协作模式。

#### Pipeline 模式（流水线）
智能体按顺序执行任务，前一个的输出作为下一个的输入。

**使用场景**：
- 文档处理流程（提取 → 分析 → 总结）
- 数据处理管道（采集 → 清洗 → 分析）

#### Graph 模式（有向无环图）
支持复杂的任务依赖关系，智能体可以并行执行。

**使用场景**：
- 复杂项目分析（多个模块并行分析）
- 多源数据聚合（并行获取后汇总）

#### Council 模式（议会）
多个智能体从不同视角分析同一问题，支持交叉审查。

**使用场景**：
- 决策分析（多角度评估）
- 代码审查（多人审查模式）
- 支持聊天窗口直接@对应智能体团队（或单个），支持飞书等频道发送文本@对应智能体团队

**新增文件**：
- `backend/api/agent_teams.py` - 团队管理 API
- `backend/models/agent_team.py` - 团队数据模型
- `backend/modules/agent/workflow.py` - 工作流引擎

### 2. 定时任务增强

#### 一次性任务
支持 `once` 类型的定时任务，执行后可自动删除。

**使用场景**：
- 定时提醒
- 延迟执行任务

#### 重试机制
- `max_retries`: 最大重试次数
- `retry_delay`: 重试延迟（秒）
- 智能错误恢复

#### 批量操作
- 批量创建任务
- 批量删除任务

**新增 API**：
- `POST /api/cron/batch` - 批量创建
- `DELETE /api/cron/batch` - 批量删除

### 4. Web 工具增强

#### Scrapling 集成
- 反爬虫能力
- JavaScript 渲染支持
- 自动重试机制

**三种抓取模式**：
- `basic`: 快速模式（1-2秒）
- `stealth`: 隐身模式（3-8秒）
- `max-stealth`: 最强隐身（10+秒）

**回退机制**：
不安装 scrapling 会自动回退到 httpx，不影响运行。

**新增文件**：
- `backend/modules/tools/web_enhanced.py` - 增强工具
- `backend/modules/tools/web_fetch_advanced.py` - 高级获取

## 升级指南

### 1. 备份数据

```bash
# 备份数据库 （与以往版本不兼容，暂未提供迁移脚本）
cp countbot.db countbot.db.backup

# 备份配置（可选）
cp -r data/ data_backup/
```

### 2. 更新代码

```bash
git pull origin main
```

### 3. 更新依赖

```bash
pip install -r requirements.txt --upgrade
```

### 4. 启动服务

```bash
python start_app.py
```

数据库会自动迁移，无需手动操作。
